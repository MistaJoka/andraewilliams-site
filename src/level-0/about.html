Why links need a base URL

Relative links like showcase.html resolve against the browser URL — not the folder on disk. This site is served from the subdomain root and from /level-0/ on the portfolio. Same HTML, two paths.

03

Request flow

  1. Request

    Browser fetches an HTML file from the host.

  2. Parse

    DOM tree built — structure before presentation or behavior.

  3. Style

    style.css applied. Layout calculated.

  4. Run

    script.js runs (defer). May fetch JSON or read storage.

04

Level 1 unlocks

Level 0 is intentionally maxed out. What you add next:

  • Custom domain at level0.andraewilliams.com
  • Full static capability showcase
  • Forms, fetch, storage, native widgets
  • Shared layouts via build step or server includes
  • Form handler — serverless function or email API
  • Generated JSON at build time from repo metadata
  • Design tokens shared with andraewilliams.com command deck

The portfolio at andraewilliams.com is Level 1+ in practice — same fundamentals, more architecture.
