02

Native HTML

Modern elements built into the browser — no component library required.

Native <dialog>

Modal focus trap, backdrop click to close, and .showModal() — all built in. This is what Level 0 looks like before you reach for a UI kit.

What is <details>?

A disclosure widget — expand/collapse without JavaScript. Used for FAQs and optional content.

Why prefer native elements?

Less code, better accessibility defaults, and behavior that survives when CSS or JS fails to load.
