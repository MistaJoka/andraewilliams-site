What Level 0 covers

Level 1 adds templating, serverless functions, and shared design systems. Level 0 is the ceiling of plain HTML, CSS, and JavaScript — every pattern below is live on this site, readable in source, and editable without compiling.

04

Level 0 checklist

  • <base href> resolver for subdomain + nested path
  • Multi-page navigation with shared CSS and JS
  • Semantic HTML, meta tags, Open Graph, canonical URLs
  • CSS Grid, Flexbox, variables, media queries, dark mode
  • Forms with native validation and accessible labels
  • fetch() loading static JSON
  • localStorage for theme preference
  • <dialog> and <details> without libraries
  • ARIA tabs, skip link, focus styles, live regions
  • Print stylesheet and prefers-reduced-motion
  • Custom 404 page
