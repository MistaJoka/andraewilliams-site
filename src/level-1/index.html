What Level 1 adds

Level 0 proved the browser can carry a whole site. Level 1 adds assembly: shared partials, repeatable page shells, and metadata generated at build time.

Partials

Header, footer, and head live once in _partials/. Pages only hold unique body content.

Build script

Generated JSON

data/site-meta.json records build time, git commit, and page list — proof the pipeline ran.
