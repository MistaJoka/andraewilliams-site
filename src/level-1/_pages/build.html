02

Build flow

  1. Read partials from _partials/
  2. For each file in _pages/, inject body between shell start and end
  3. Write assembled HTML to src/level-1/*.html
  4. Emit data/site-meta.json with timestamp and git commit
  5. assemble-site.sh copies everything to _site/ for deploy
03

Run locally

node scripts/build-level-1.mjs
npm run build:site
npm run preview:site

Edit _pages/ or partials, rebuild, refresh. No framework required.
