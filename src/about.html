About

// origin story

This is my domain, my lab, my playground — a sci-fi inspired workspace where I build custom tools, apps, and experiments to sharpen real skills.

01 · mission

Build in public

andraewilliams.com is a living portfolio: proof of what I can do, documented step by step. I start with static tools, then grow toward server-based apps as the concepts click.

02 · vibe

Sci-fi, but real

Nostalgia for the tech I grew up admiring — command centers, glowing interfaces, the feeling of a personal Batcave — built with today's stack and zero fluff.

03 · constraints

Free tools only

MacBook, GitHub, GitHub Pages, and free tiers. Every project has to work within that box until there's a clear reason to expand.

04 · path forward

What's next

More tools in the library. Stronger interactions. Eventually access control and server-backed apps — learning security and backend logic the same way: one small, working slice at a time.
