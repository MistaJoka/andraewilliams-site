# Project Brief

## Project
andraewilliams.com

## Goal
To build a website that showcases my technical skill. This website is my domain, my lab, my playground. While public on the internet, it's just for me and those I invite. I want it to remind me of my passion for computers and love for sci-fi tech. This place if my Batman lair, my IronMan, my Superman AI. We start small with static apps and tools added to the website. Then over time we progress to server based apps as I continue to show my understanding of the logic and concepts.

## Audience / User
Me mainly. I would be visiting the site mostly. Unless someone finds it online. I would occasionally tell others about it when excited or when I would like to demonstrate a project. I am considering access control to showcase cybersecurity skills. preferably minimal but resillient measures. 

## Success Criteria
- Functional website
- Cool looking. nostalgia enducing. Sci-Fi tech inspired but hardcoded in todays modern reach. 
- Library of tools, apps, and games. All custom built by me and added to my website.
- Overall is an example of what I can do. My online portfolio. I would show this to a close friend or potential client my personal passion side project.
- Along the way I want to log and document the incremental changes made. 

## Constraints
- Budget: Minimal as possible until additional capabilites are needed.
- Tech limits: my main workstation is a macbook pro, M5 Chipeset, 16GB RAM.
- Other limits: website domain is managed in GoDaddy. Code held in GitHub. Vercel is used to enable one of my apps at promptbook.andraewilliams.com
- I only want to make this project possible by using free tools, and my paid ChatGPT subscription(and everthing it offers including Codex and Canvas). I am willing to use other tools if they are free.  

## Scope
### In scope
- Projects that will work alongside my current setup. 
- Ideas that add to the website's functionality, maintainability, scalability, and future proofing. 

### Out of scope
- Projects or ideas that require a Gradulate level education. I have years of reading basic python code to where i can understand the logic, but i lack application to have a full cohesive knowledge set. 
- Steps that require me to pay or have temporary access to a third party tool.
